# 文章正文插图说明

文章发布工具现在可以按照光标位置插入图片，不需要手写 Markdown 路径。

## 使用方法

1. 双击 `启动文章发布工具.bat`，导入原稿或继续已有草稿。
2. 点击“编辑正文并插图”。
3. 在正文中点击希望出现图片的位置。
4. 点击“在光标处插入图片”，选择 PNG、JPG 或 JPEG。
5. 填写一句简短、真实的图片说明，例如“Vercel 部署成功结果”。
6. 保存后点击“检查草稿”和“浏览器预览”。
7. 确认无误后正式上线，工具会同时提交当前文章和它的图片。

## 工具自动处理的内容

- 图片最长边超过 1600 像素时自动缩小。
- JPG 使用 85% 质量重新编码；PNG 保留透明背景。
- 重新绘制图片以移除原文件中的 EXIF 等元数据。
- 使用安全文件名，保存到当前文章专属的草稿图片目录。
- 草稿预览时临时映射图片，停止预览后自动清理。
- 发布时将图片移动到 `public/images/articles/<slug>/`。
- 完整检查失败时恢复 Markdown 和草稿图片，不留下半发布内容。

## 发布前仍需人工确认

工具不会判断截图里是否包含隐私。正式上线前，请检查姓名、邮箱、IP、Token、Cookie、订单和账号等敏感信息。

第一版只支持 PNG、JPG 和 JPEG，不包含封面图、图床、灯箱或富文本排版。
