## 项目简介

这是一个用于实验 AI Agent 架构的实验框架。目的是通过实践来理解不同的 Agent 设计模式。

### 特性

- 支持多种 LLM 后端（OpenAI、Ollama）
- 基于 LangGraph 的工作流引擎
- 模块化的工具系统
- 实验日志记录
- 简单的 Web 界面

### 设计思路

项目采用插件式架构，每个 Agent 组件都是可插拔的模块。这样可以方便地切换不同的实现，对比不同方案的优劣。
