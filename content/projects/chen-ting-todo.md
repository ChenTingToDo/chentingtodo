## 项目简介

ChenTingToDo 是一个数字花园（Digital Garden）风格的个人知识管理平台。

它不是一个传统博客，而是一个长期成长记录平台。

### 核心理念

- **Think Less. Do More.** — 行动优于完美
- **Build In Public** — 公开记录学习过程
- **Knowledge Compounding** — 知识复利
- **Personal First** — 首先服务自己

### 技术栈

- **框架**: Next.js 14 (App Router)
- **样式**: Tailwind CSS
- **内容**: Markdown + Frontmatter
- **搜索**: Pagefind
- **部署**: Vercel

### 功能

- ✅ 文章发布与管理
- ✅ 项目展示
- ✅ 全文搜索
- ✅ 深色模式
- ✅ 响应式布局
- ✅ SEO 优化
- ✅ GitHub 管理
- ✅ 自动部署
